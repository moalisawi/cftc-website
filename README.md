# 🌐 CFTC — Canadian Foundation for Training & Consulting
### المؤسسة الكندية للتدريب والاستشارات

موقع ويب تسويقي احترافي باللغة العربية (RTL) للمؤسسة الكندية للتدريب والاستشارات — شركة استشارية فلسطينية متخصصة في شهادات ISO المعتمدة دولياً.

## 🚀 تشغيل الموقع

### الطريقة الأسرع — فتح مباشر
```
افتح index.html في أي متصفح
```

### تشغيل على سيرفر محلي (موصى به)
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .

# ثم افتح: http://localhost:8080
```

## 📦 الأقسام

| القسم | الوصف |
|-------|-------|
| **Hero** | عنوان رئيسي + بطاقة مراحل ISO + 5 إحصائيات بعدّاد متحرك |

| **العملاء** | 17 عميل مع فلاتر حسب القطاع + شعارات |
| **الباقات** | 3 خطط تسعير (عرض مخصص) |

| **Footer** | روابط + معلومات الشركة |

## ✨ المميزات التقنية

- **Sticky Header** — شفاف مع blur effect
- **Mobile Navigation** — قائمة منسدلة للجوال
- **Counter Animation** — عدّاد الإحصائيات عند الظهور
- **Service Tabs** — 5 تبويبات تفاعلية لعرض الخدمات
- **Client Filters** — فلترة العملاء حسب القطاع
- **Active Nav** — تتبع القسم النشط عند التمرير
- **WhatsApp FAB** — زر واتساب عائم مع رسالة جاهزة
- **Sticky Mobile CTA** — شريط ثابت في الجوال
- **Exit Intent Popup** — نافذة تظهر عند محاولة المغادرة
- **Contact Form** — نموذج مع تحقق وتأكيد
